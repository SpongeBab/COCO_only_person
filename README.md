# COCO_only_person
Use the python script to select images contains person in the COCO。

使用这个脚本去挑选出coco数据集中只包含人的图片。
## Note
注意：你需要以下文件。
1.coco2017数据集,训练集和测试集。（）
2.txt格式的标签文件。（）
3.Python环境。

通过txt标签文件判断当前图片是否包含人，所以txt文件是必须的。
